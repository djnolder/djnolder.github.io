My Works

Long Lake LTD

Site was built with relationships between Communities, Builders, Plans and Inventory. Pricing is based on these relationships and can very drastically. A custom plugin was built to handle the custom post types along with their relationships. One critical feature being able to have variable pricing based on these relationships. For example, even though the plans are the same between two communities, the prices for those plans is not. In the past, they had a page for each plan in a community, and would manually update those pages. To handle this in the new relationship system a custom database was created for pricing. A custom model was created for this database, as well as an admin tool to manage it and a custom shortcode for outputting prices. What used to take the client days to manage was reduced to minutes.
